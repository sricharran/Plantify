# Plantify

## Online Plant Shopping Cart Application

Plantify is a React-based front-end shopping cart application for an online plant store. The application allows users to browse a variety of houseplants, add plants to a shopping cart, manage item quantities, and view the total cost dynamically.

## Project Overview

This project was developed as a final front-end project to demonstrate skills in React, component-based architecture, state management, and dynamic UI updates.

## Features

- Landing page with a brief introduction and a Get Started button
- Product listing page with categorized houseplants (e.g., Aromatic Plants and Medicinal Plants)
- Plant cards displaying image, name, description, and price
- Add to Cart functionality
- Shopping cart page with:
  - Increase and decrease quantity buttons
  - Delete item option
  - Dynamic total item count and total price
- Navigation bar with real-time cart quantity updates

## Technologies Used

- React
- Vite
- Redux Toolkit
- JavaScript (ES6)
- HTML & CSS

## Deployment

The Plantify project is hosted in a public GitHub repository and deployed using GitHub Pages.
